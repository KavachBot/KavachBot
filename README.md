## Overview

**Kavach** — meaning *armor* in Sanskrit — is a battle-tested Discord security bot built to defend servers against raids, nuke attempts, unauthorized mass actions, and malicious actors. Available in both a free and premium tier, Kavach scales to protect communities of any size.

---

## Features

<table>
<tr>
<td width="50%" valign="top">

**AntiNuke Protection**
Detects and neutralizes mass channel deletions, role modifications, and permission escalations in real time before damage is done.

<br/>

**Anti-Raid System**
Identifies coordinated join floods and automated bot raids. Automatically locks down entry and alerts administrators instantly.

<br/>

**Automated Moderation**
Intelligent auto-mod covering spam, invite links, mass mentions, caps abuse, and fully customizable word filters.

</td>
<td width="50%" valign="top">

**Backup & Restore**
Complete server snapshots — channels, roles, permission overwrites — with one-command restore capability.

<br/>

**Verification Gates**
Captcha-based and button-gate entry systems that block bot accounts and unverified users from accessing your server.

<br/>

**Audit Logging**
Granular, timestamped logs for every moderation action, member join/leave, role assignment, and permission change.

</td>
</tr>
</table>

## Getting Started

**1.** Invite Kavach to your server using the invite link above.

**2.** Run the setup command and configure your desired protection level.

**3.** Assign Kavach a high-ranked role so it can act on threats effectively.

**4.** Kavach handles the rest — silently, in the background, always watching.
